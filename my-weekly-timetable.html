HTML Topics Reviewed

Table Elements

  • <table> - Creates the table structure
  • <thead> - Table header section
  • <tbody> - Table body section
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell
  • colspan - Attribute to span multiple columns

Other HTML Elements Used

  • Headings: <h1> through <h4> for document structure
  • Paragraphs: <p> for text content
  • Lists: <ul>, <ol>, and <li> for organized information
  • Links: <a> with href, target, and rel attributes
  • Semantic HTML: <header>, <main> for better structure
  • Text Formatting: <strong>, <em>, <span> for emphasis
  • Line Breaks: <br> for line breaks within cells
  • Code: <code> for displaying code elements
