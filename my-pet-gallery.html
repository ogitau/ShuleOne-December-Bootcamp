Understanding Relative Paths

  • pet1.jpg or ./pet1.jpg - Image in the same directory as the HTML file
  • images/pet2.jpg or ./images/pet2.jpg - Image in a subdirectory called "images"
  • assets/images/pet5.jpg - Image in a nested subdirectory structure
  • ../pet6.jpg - Image in the parent directory (one level up)
  • ../../pet.jpg - Image two levels up from current directory

Note: The images above use placeholder URLs. To use your own images, place them in the appropriate directories and update the src paths accordingly.
